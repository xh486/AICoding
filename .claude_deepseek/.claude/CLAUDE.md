# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述
这是一个 Claude Code 配置仓库，用于存放 Claude Code 的 settings、workflows、skills 等配置项。

## 目录结构
- `.claude/` — Claude Code 项目级配置（settings.local.json、hooks、workflows 等）
- 用户级 CLAUDE.md 已包含通用规则（沟通方式、Git 规范、红线操作），此处不再重复

## 注意
- settings.local.json 包含项目级权限配置，修改需谨慎
- 此仓库不包含应用代码、无构建/测试/lint 命令
